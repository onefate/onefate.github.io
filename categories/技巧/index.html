第一步：安装 GIt for Windows、nodejs 以及安装 Hexo 博客系统

  1. Git for Windows 以及 nodejs 就看做是两个必要的软件吧，下载并安装好。网盘直接下载
  2. 安装完成之后，系统右键菜单里就会多了几个选项。要用到的是 Git Bash 选项。在任意位置右击选择 Git Bash。输入以下命令，这就是要安装 Hexo 博客：

第二步：Github 注册、Gitcafe 注册

  1. Github 和 GitCafe 任选其一。存放博客文件的地方，想象成网盘。

第三步：执行 hexo 命令

  1. 在电脑的任意位置新建一个文件夹，进入这个文件夹。再次地点击鼠标右键选择 Git Bash。
    （注意：一定要在新建的文件夹内：右击——选择 Git Bash 哦）
  2. 输入 Hexo init ，回车，这是初始化的意思。没意外的话，就会在本文件夹内生成一些文件了。
  3. 此时雏形已经有了，并且可以试着看看效果。
  4. 输入 Hexo g，回车，这是生成博客页面。但是要如何访问的到呢？
  5. 输入 Hexo s，回车，这是把生成的内容挂到本地的端口上。这样子我们就可以用服务器访问了。
  6. 打开游览器输入：http://localhost:4000/，如果看到一篇名为“Hello World”的文章，那就说明之前的操作都是有效地。
